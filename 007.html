법령 개정조문 올리는 방법 순서 [법령 개정 정보 -> 법령 개정 조문]

1. 법령개정정보

*개정된 법령의 전문, 신구대조표, 개정요약, 개정된별표를 확인하고 한글파일은 PDF로 변환하여 각각 한글파일 1개 PDF파일 1개를 만든다.
*유니티엠 로그인 - 오른쪽 상단 이름 클릭 - 사이트 관리 - 법령개정관리 - 법령개정정보 클릭 - 법령 찾아서 클릭 - 법령개정번호 추가 - 동법개정, 타법개정 확인 후 고시일자, 시행일자(확정으로 바꿔야함) 입력 후 저장 - 아래 고시내용 클릭 (전문, 개정요약, 신구대조표 하나씩 추가후 오른쪽에 목록이 뜨면 파일올리기로 각각 한글파일과 pdf 파일 업로드(전송), 기타 파일첨부시 기타로 추가후 제목을 별도입력 *법령정보를 모두 업로드 후 목록 - 첨부서류 - 별표 파일명 추가 (별지서식등은-서식, 벌표는 별표로) 전체별표인 경우 개정구분-현재, 개정별표의 경우 -개정,신설,삭제로 맞게입력(삭제일 때는 파일명 삭제로하고 파일 올리지 않음, 시행일자가 다른경우 시행일자 고쳐서 입력) 파일명 추가후 파일 업로드 - 원본저장 체크 클릭후 완료로 저장함.

2.법령조문관리

* 법령관리 - 개정조문관리 - 개정조문관리 - 법령 클릭후 확인 - 법령개정구분 작업줄의 고시원본가져오기 - 신구대조표 이면지출력 - 전문은 QR코드 없애고 불러올수 있는곳에 저장 -관리작업클릭 - 파일올리기 (전문 QR없앤것) - 조문체계 (테스트 NO 재작업 YES) 로 한번전송, 조문(테스트 NO 재작업 YES)로 한번 전송 - 왼쪽 목록에서 본문과 부칙이 맞게 올라왔는지 신구대조표, 전문과 비교 - 맞지 않는경우 맞게 고치고 맞게 올라간경우 위쪽법령개정 클릭 - 목록 - 조문체계와 저문 전송하면 입력일자 자동체크 되지만 교정일자는 개정내용 맞게 전송되었는지 확인 후 체크( 오류나는경우 결과조회에 들어가서 오류 난 내용 수정 후 다시 체크) - 체크시 오늘 날짜가 뜨면 된것 - 오른쪽에서 종료일 재설정 클릭 - 종료일 설정 - 닫기 - 조문데이터 확정 - 재작업 미확정 둘다 YES 로 바꾼후 확정 - 닫기

*조문 입력시 체크사항

1. 전문 법령 정문 부칙에서 시행일자 확인하여 개정되는 법의 시행일자에 체크함.
(보통은 부칙에 시행일자가 하나이지만 조의 개정내용마다 시행일자 다를수 있음)
2. 항이나 호의 삭제 - 개정구분 삭제의 경우 항이나 호의 중간에 삭제일 경우에는 종료일 설정하지 않음( 종료일 설정은 항이나 호가 안보이게 삭제시 종료설정)
3. 1항의 삭제일 경우 조를 현재로 명칭만 넣어 입력하고 1항을 삭제하여야함.
4. 조의 삭제 - 조의 시행일자만 입력하고 종료일자는 비워둠 ( 나머지 항 호 등은 종료일 설정해야 조만 삭제로 보임.)
5. 전문내용에 표가 올라와있는경우 HTML로 작업해서 표를 직접 만들어줘야함.
6. 부칙 - 조:부칙조, 개정구분 항상 현재 명칭 - 타법은 호와 개정일자 타법제목 (예:<제415호, 2021.4.1 (전기안전관리법 시행규칙)) 동법은 개정호와 개정일자 (예:<415호, 2021.4.1>) 내용 - 타법은 텍스트 파일로 부칙내용 모두입력, 동법은 본문과 같이 조별로 입력함.
